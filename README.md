# FritoMapp

## Descripción

Aplicación web interactiva para el **XLII Festival del Frito Cartagenero**. Permite a los asistentes explorar los puestos de las matronas, votar por su frito favorito y seguir el ranking en tiempo real.

## Funcionalidades

- **Directorio de puestos** — Lista de matronas con búsqueda y filtros por categoría
- **Perfil de matrona** — Biografía, especialidad, galería de fritos y ubicación en mapa
- **Sistema de votación** — Un voto por usuario, puede cambiarlo cuando quiera
- **Top 5 en tiempo real** — Ranking actualizado automáticamente con Supabase Realtime
- **Galería multimedia** — Imágenes de los fritos con lightbox y filtros por categoría
- **Salón de la Fama** — Ganadores históricos agrupados por edición
- **Códigos QR** — Generados automáticamente por puesto para acceso directo al perfil
- **Mapa interactivo** — Ubicación exacta de cada puesto con Leaflet + OpenStreetMap

## Tecnologías

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js + Express
- **Base de datos:** Supabase (PostgreSQL)
- **Storage:** Supabase Storage
- **Mapas:** Leaflet + OpenStreetMap
- **Hosting:** Railway
